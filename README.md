Relatorio-de-Conjuntura

O Relatório sofreu algumas mudanças de metodologia ao longo do tempo. Primeiramente, as análises e previsões eram feitas através do modelo IS/LM (Hicks 1937). A partir da impossibilidade de se prever com IS/LM, as previsões começaram a ser feitas através do modelo ARIMA (Box, Jenkins 1970), enquanto a análise ainda era feita através do IS/LM. Finalmente, a partir da incongruencia de utilizar um modelo para a análise e outro para as previsoes, ambas passaram a serem feitas através de modelo VAR (Sims 1980).
