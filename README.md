Relatorio-de-Conjuntura

O Relatório sofreu algumas mudanças de metodologia ao longo do tempo. Primeiramente, as análises e previsões eram feitas através do modelo IS/LM (Hicks 1937). A partir da impossibilidade de se prever com IS/LM, as previsões começaram a ser feitas através do modelo ARIMA (Box, Jenkins 1970). Finalmente, a partir da incongruência de se utilizar um modelo para a análise e outro para as previsões, ambas passaram a ser feitas através de modelo VAR (Sims 1980).

The Report has undergone some changes of methodology over time. First, the analyzes and predictions were made using the IS / LM model (Hicks 1937). From the impossibility of predicting with IS / LM, the predictions began to be made through the ARIMA model (Box, Jenkins 1970). Finally, from the incongruity of using one model for the analysis and another one for the predictions, both started to be done through VAR model (Sims 1980).
